A modern, responsive **Kanban Board** built using **React**, **Context API**, **TailwindCSS**, and **dnd-kit**.  
This application allows users to manage tasks efficiently using drag-and-drop functionality with persistent storage via `localStorage`.


#  Features

-  Create tasks with title and description  
-  Edit task description and status via modal  
-  Delete tasks  
-  Drag and drop tasks between columns  
-  Persistent storage using `localStorage`  
-  Fully responsive UI  
-  State management using React Context API  

# Kanban Columns

- **To Do**
- **In Progress**
- **Done**

Tasks can be freely moved across columns using drag-and-drop.

---

# Tech Stack

- **Frontend:** React JS (Vite)
- **State Management:** Context API
- **Styling:** TailwindCSS
- **Drag & Drop:** @dnd-kit/core, @dnd-kit/utilities
- **Storage:** Browser localStorage
- **Utilities:** UUID
